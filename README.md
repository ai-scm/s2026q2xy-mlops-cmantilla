# Ejercicio MLOps y AIOps

**Autora:** Carolina Mantilla
**Fecha:** 1 de junio de 2026

## 📌 Descripción

Este repositorio contiene el desarrollo del ejercicio sobre **MLOps y AIOps**.
El objetivo es comprender el ciclo de vida de Machine Learning desde una perspectiva operativa: datos, modelos, software y monitoreo.

Para este trabajo se seleccionaron dos lecturas:

* **3. End-to-End ML Workflow Lifecycle**
* **7. MLOps Stack Canvas**

## 🧠 Contenido

### 1. End-to-End ML Workflow Lifecycle

Explica el ciclo completo de un proyecto de Machine Learning: datos, entrenamiento, evaluación, despliegue, monitoreo y mejora continua.

### 2. MLOps Stack Canvas

Resume las capas y herramientas necesarias para operar modelos de Machine Learning en producción.

### 3. MLOps vs AIOps

Define AIOps y explica su relación con MLOps.

## 🔗 Relación con MLflow

Este ejercicio se conecta con el taller anterior de **MLflow**, ya que MLflow es una herramienta usada dentro de MLOps para registrar experimentos, versionar modelos y apoyar su despliegue.
